What I do

Automation Scripting

Automation of system processes, building reports from databases, extracting data from files, building reports in Excel, tasks on networking devices over SSH and extracting data from web resources like websites or REST API's.. Scheduling scrips to run on SMA OpCon, Windows Task Scheduler or in a custom timer. I can also update your code from Python 2 to Python 3.

Web Development

I specialize in building web applications in Flask, SQLAlchemy, HTML, CSS and Javascript. I also use other frameworks and resources like Bootstrap, Slick and Font Awesome. I can host the application on Amazon Beanstalk, PythonAnywhere, Windows IIS, Heroku or Apache servers. To store data I use PostgreSQL, Windows SQL, SQLite or MySQL.

GUI Development

I can build Desktop applications that are cross-platform and run on Windows, Linux or Mac OS using Tkinter or QT4. I can also build Windows only Desktop applications that use .NET using Iron Python and SharpDevelop for faster development. The desktop applications can use any Database of your choice and include setup files for installations or single portable executables.

Amazon AWS

The desktop and web applications I build can use Cloud resources like Amazon S3 to store your files and images, Amazon RDS where you can have your databases with your data, Amazon EC2 to host your web applications or manage your cloud servers. I can also manage the security and security policies of your Amazon account and other cloud resources.
